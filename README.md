# Express.Js-Typescript-Bun

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run start
```

To build:

```bash
bun run build
```
Start/stop database:
```bash
bun run db:start
bun run db:stop
```
Create table
```bash
bun migrate up
```

This project was created using `bun init` in bun v1.1.4. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
